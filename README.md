# Dashboard de Análise de Vendas

## Objetivo
Analisar o desempenho de vendas de uma empresa fictícia, avaliando receita, custo e margem por produto, grupo e vendedor.

## Ferramentas
- Power BI
- Modelagem de dados

## Principais Análises
- Receita, custo e margem total
- Evolução da receita ao longo dos meses
- Margem por linha e grupo de produto
- Desempenho de vendedores e supervisores

## Insights
- A linha de produtos Bebidas apresenta maior margem acumulada.
- Alguns vendedores possuem desempenho significativamente superior à média.
- Há grupos de produtos com alta receita, porém menor margem.

## Conclusão
O dashboard auxilia gestores na tomada de decisão ao evidenciar oportunidades de melhoria na rentabilidade e no desempenho comercial.
